# 🚀 SuiYuan Search 项目介绍
## 📋 项目概述
SuiYuan Search （随缘起始页）是一个简洁、开源、可自定义的浏览器扩展，专为用户提供个性化的新标签页搜索体验。该项目集成了多种搜索引擎、丰富的主题定制功能和流畅的用户界面。

## ✨ 核心特性
### 🔍 多搜索引擎支持
- 内置搜索引擎 ：必应、谷歌、百度等主流搜索引擎
- 自定义搜索引擎 ：用户可添加个性化的搜索引擎
- 智能切换 ：支持快速切换不同搜索引擎
- 图标管理 ：自动获取或自定义搜索引擎图标
### 🎨 丰富的主题定制
- 双主题模式 ：亮色主题 / 暗色主题 / 跟随系统
- 多样化背景 ：
  - 默认背景
  - 纯色背景（支持自定义颜色）
  - 渐变背景（支持方向和颜色配置）
  - 图片背景（支持透明度和模糊效果）
  - 视频背景（支持透明度和模糊调节）
- 可编辑标题 ：支持自定义页面标题文字
